# First Person Shooter: Wallhack

## Wallhack
We are making a FPS game called Wallhack. The game will provide basic FPS functionallity for multiplayer. The key goal how ever of this game is to explore how to make FPS games
more secure in terms of wallhack exploits. This will be achieved by enabling the client to show the locations of all other players and having the server share as little
as possible with the clients, making wallhacks more or less impossible. Location updates should only occure when the players can see eachother.
